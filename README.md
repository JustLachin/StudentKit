## Hakkında | About
StudentKit, öğrenciler için geliştirilmiş kapsamlı bir üretkenlik uygulamasıdır.  
StudentKit is a comprehensive productivity application developed for students.

## Özellikler | Features
- **Pomodoro Zamanlayıcı**: Odaklanma seansları ile verimli çalışma  
  **Pomodoro Timer**: Work efficiently with focused sessions  

- **Not Sistemi**: Ses kayıtlı ve metin notları  
  **Notes System**: Voice recordings and text notes  

- **Haftalık Ders Programı**: Ders takibi ve program yönetimi  
  **Weekly Schedule**: Track classes and manage schedules  

- **Sınav Takvimi**: Sınav tarihleri ve geri sayım  
  **Exam Calendar**: Exam dates and countdown
